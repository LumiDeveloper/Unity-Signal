# Signal Roguelite — как запустить проект

Привет! Это репозиторий с игрой для Ludum Dare.  
Ниже инструкция, как скачать и запустить проект без единой ошибки.

---

## 📦 Что нужно иметь

- **Unity Hub** (последняя версия)
- **Unity 2022.3.62f3** (у тебя именно такая, всё подойдёт)

> У меня версия 2022.3.50f1 — разница в минорной версии не страшна, Unity сама всё подтянет.

---

## 🚀 Инструкция (3 минуты)

### 1. Скачай репозиторий

**Способ А — через GitHub Desktop (проще):**
- Нажми `Code` → `Open with GitHub Desktop`
- Выбери папку, куда сохранить

**Способ Б — ZIP-архивом:**
- Нажми `Code` → `Download ZIP`
- Распакуй в любую папку

---

### 2. Открой проект в Unity Hub

1. Запусти **Unity Hub**
2. Нажми кнопку **Add** → **Add project from disk**
3. Выбери **папку**, которую скачал (ту, где лежат папки `Assets`, `Packages`, `ProjectSettings`)
4. Проект появится в списке

---

### 3. Запусти

- Нажми на проект в Unity Hub
- Unity сама:
  - Подгрузит нужные пакеты
  - Создаст папку `Library` (это нормально, она не в репозитории)
  - Откроет сцену

> ⏳ Первый запуск может занять 2–3 минуты — Unity настраивается.

---

### 4. Играй

- Нажми **Play** в редакторе
- Выбирай цвета и фигуры
- Шифруй сигнал и защищай его от врагов

---

## ⚠️ Если что-то пошло не так

| Проблема | Решение |
|----------|---------|
| Ошибка про `Library` | Это нормально, папка создаётся при первом запуске |
| Красные ошибки в консоли | Закрой Unity, удали папки `Library` и `Temp`, открой заново |
| Не открывается сцена | В окне `Project` → `Assets/Scenes` → кликни `SampleScene` дважды |

---

## 🎮 Управление (всё мышкой)

- Выпадающие списки — выбирай цвета и фигуры
- «Разведка» — посмотреть врага
- «Отправить» — запустить битву
- «След. уровень» — после победы
